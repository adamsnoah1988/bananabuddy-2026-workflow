## What BananaBuddy Does

BananaBuddy is built to bring repetitive automation into a single, organized flow. Instead of handling isolated scripts one by one, it helps manage Telegram-based processes, chained tasks, and related actions in a more structured way so they can be repeated, tuned, and observed over time.

The framework also emphasizes visibility and flexibility. With support for a web dashboard, multiple languages, and optional AI integrations, it can adapt to different setups while still giving users a practical view of what is running and how resources are being used.

## FAQ

**How do I stay current?**  
Use the latest release or build link in this repository and replace your local copy with the newest package when a new one is available.

**Where do I change settings?**  
Depending on the version you are running, update them in the project configuration files or in the dashboard.

**What if a task behaves unexpectedly?**  
Check the workflow definition, verify the input state, and refine the interaction rules for the step that is failing.

**Can it run without AI services?**  
Yes. Core automation can be structured around your environment, while OpenAI or Claude support is only used when you enable those integrations.

**Who is this project for?**  
It is intended for users who need organized automation for Telegram-related or task-focused workflows and want a dashboard-based way to manage them.
